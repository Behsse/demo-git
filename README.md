# Demo d'un projet git

## Présentation :
Ce projet est géré par git.

## Commande git
* Pour intialiser git dans un dossier :
```
git init
```

* Voir les changements depuis la dernière sauvegarde :
```
git status
```

* Ajouter les dernières modifications du fichier `README.md`
```
git add README.md
```
